Crkyll 0.5.0 -- Crkyll is a blog-aware, static site generator in Crystal.

```
Usage:
  ./bin/crkyll [flags...] [arg...]

Crkyll 0.5.0 -- Crkyll is a blog-aware, static site generator in Crystal.

Flags:
  --help        # Displays help for the current command.

Subcommands:
  build, b      # Build your site
  clean         # Clean the site (removes site output and metadata file) without building.
  compose
  docs
  doctor, hyde  # Search site and print specific deprecation warnings
  import
  new           # Creates a new Crkyll site scaffold in PATH
  new-theme     # Creates a new Crkyll theme scaffold
  server, s     # Serve your site locally
```
